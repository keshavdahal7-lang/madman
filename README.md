# Madman Form Demo Project

## Run Locally

1. Install dependencies:
   `npm install`
2. Start development server:
   `npm run dev`
3. Build production output:
   `npm run build`

## Demo Routes

- `/demos` : Demo index page with links
- `/demos/1` : Icon-Pattern Hero
- `/demos/2` : Feature Grid
- `/demos/3` : Performance Dashboard
- `/demos/4` : Premium Landing
- `/demos/5` : Minimal Luxe

## Notes

- Existing palette is preserved and reused across all demos.
- Reusable fitness icon system lives in `src/components/FitnessIcons.tsx`.
- Core content is enhanced, not removed.
